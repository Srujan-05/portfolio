- [ ] Update phone number
- [ ] Update email addresses
- [ ] Update about me paragraph
- [ ] Add your research interests
- [ ] Add your technical skills

Edit **`src/pages/Projects.jsx`**:
- [ ] Add your projects (replace the 4 sample projects)
- [ ] Include project title, description, tech stack, GitHub link
- [ ] Add at least 2 projects (4 is better)

### Step 2: Test Locally (5 minutes)

```bash
cd "Assignment-4 Portfolio/portfolio-app"
npm install
npm start
```

- Portfolio opens at http://localhost:3000
- Click around, test navigation
- Check mobile view (F12 → Device Toolbar)
- Press Ctrl+C to stop

### Step 3: Deploy (5 minutes)

```bash
# Update package.json homepage
# Change "https://your-username.github.io/portfolio" to YOUR GitHub username

# Deploy to GitHub Pages
npm run deploy
```

Your portfolio is live at:
```
https://yourusername.github.io/portfolio
```

## ❓ Common Questions

**Q: Can I change the colors?**
A: Yes! Edit `:root` in `App.css` to customize the entire theme.

**Q: How do I add more projects?**
A: Edit the `projects` array in `Projects.jsx`. Just add another object!

**Q: Will it work on mobile?**
A: Yes! Fully responsive design. Test with F12 → Device Toolbar.

**Q: Can I deploy to a custom domain?**
A: Yes! GitHub Pages supports custom domains. See their documentation.

**Q: How do I update after deployment?**
A: Edit files → git commit → git push → npm run deploy. Done!
